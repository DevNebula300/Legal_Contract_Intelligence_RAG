LegalIntelligence

A full-stack application for legal contract intelligence. It allows users to upload contracts, identify legal risks, compare agreements, search precedents, and generate professional legal reports.

Getting Started

Frontend

1 Navigate to the frontend directory.
2 Install dependencies using npm install.
3 Start the development server using npm run dev.
4 Open http://localhost:3000 in your browser.

Backend

1 Navigate to the backend directory.
2 Install dependencies using pip install -r requirements.txt.
3 Start the FastAPI server using uvicorn app.main:app --reload.
4 The API will be available at http://localhost:8000.

Features

Upload and Analyze: Automatically parse and classify legal documents.
AI Chat: Ask questions about the uploaded contract and get citations.
Risk Review: AI identifies high, medium, and low risk clauses.
Precedent Search: Find relevant court cases and legal precedents based on contract clauses.
Compare Contracts: Compare two versions of a contract to highlight differences clause by clause.
Generate Report: Generate and view a comprehensive risk review report.
